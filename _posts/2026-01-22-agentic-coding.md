## 從 Copilot 到 Agent

過去我們使用 Copilot，比較像是遇到問題去查 Stack Overflow 的進化版——它給你片段的代碼。但現在，隨著 Agent Manager 這類工具的出現，遊戲規則改變了。

### 什麼是 Agentic Mode？

在 Agentic Mode 下，我不再是輸入單一指令，而是給出一個「目標 (Objective)」。例如：「幫我把這個 Repo 的部落格功能建立起來，包含五篇文章」。

Agent 會做什麼？
1.  **規劃 (Planning)**: 掃描檔案結構，列出待辦清單 (Task List)。
2.  **執行 (Execution)**: 自動修改 `_config.yml`，生成 Markdown 檔案。
3.  **驗證 (Verification)**: 檢查檔案是否正確生成。

### 人類角色的轉變

在這個模式下，我們從 **Coder** 變成了 **Product Manager** 或 **Lead Engineer**。我們負責審核 (Review) 計畫，負責定義邊界 (Boundary)，而將繁瑣的實作交給 Agent。這是我近期體驗到最高效的工作流。
